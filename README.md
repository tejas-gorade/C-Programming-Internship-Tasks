# Internship Project Portfolio (C Programming)

This repository contains the core tasks completed during my 4th-semester internship. 
The focus was on applying Data Structures and Systems Programming concepts to real-world scenarios.

## 🚀 Projects Overview

| Project | Key Concepts | Why it matters? |
| :--- | :--- | :--- |
| [**Inventory System**](./inventory-system) | File I/O, Structs, Persistence | Shows ability to manage permanent data. |
| [**Playlist Manager**](./music-playlist-manager-code) | Linked Lists, Dynamic Memory | Demonstrates pointer safety and heap management. |
| [**Lexical Cleaner**](./whitespace-comments-remover) | Parsing, State Logic | Highlights low-level code processing skills. |
| [**RLE Compressor**](./RLE-Compressor-code) | Algorithms, Data Efficiency | Proof of algorithmic thinking and optimization. |

## 🛠️ Optimization & Quality
Each project was initially developed as a functional prototype and subsequently refined to implement:
- **Robust Input Validation:** Handling edge cases in user input to prevent crashes.
- **Memory Safety:** Ensuring no memory leaks or dangling pointers using `free()`.
- **File Integrity:** Proper error handling for file streams and EOF safety.

## ⚙️ Setup & Execution
1. Clone the repo: `git clone https://github.com/tejas-gorade/C-Programming-Internship-Tasks.git`
2. Navigate to any task folder.
3. Compile using GCC: `gcc filename.c -o output`
4. Run the executable: `./output`
