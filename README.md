# Internship Project Portfolio (C Programming)

This repository contains the core tasks completed during my 4th-semester internship. 
The focus was on applying Data Structures and Systems Programming concepts to real-world scenarios.

## 🚀 Projects Overview

| Project | Key Concepts | Why it matters? |
| :--- | :--- | :--- |
| **Inventory System** | File I/O, Structs, Persistence | Shows ability to manage permanent data. |
| **Playlist Manager** | Linked Lists, Dynamic Memory | Demonstrates pointer safety and heap management. |
| **Lexical Cleaner** | Parsing, State Logic | Highlights low-level code processing skills. |
| **RLE Compressor** | Algorithms, Data Efficiency | Proof of algorithmic thinking and optimization. |

## 🛠️ Optimization & Quality
Each project was initially developed as a functional prototype and subsequently refined using **AI-assisted code reviews** to implement:
- **Robust Input Validation:** Handling edge cases in user input to prevent crashes.
- **Memory Safety:** Ensuring no memory leaks or dangling pointers.
- **File Integrity:** Proper error handling for file streams and EOF.
